\section*{Test Procedure}
\begin{enumerate}[noitemsep]
    \item Create the testing environment
        \item Place one piece of tape(12 ft) on the floor
        \item Place a second piece of tape extending 8 ft from the first at a right angle at the left end of the first piece of tape. The 
        \item place another piece of tape at a right angle to the first piece of tape using a t-square, to the left side of the tape. The first piece of tape is $x=0$ and positive x is in the direction of the second piece of tape.
    \item Place robot in the nominal resting position
    \item Complete pretest checklist
        \item Check the pressure of each tube, if they are not at 15 psi, inflate them to 15 psi
        \item Check each joint and ensure it has a pin and is securely connected
        \item check charge of each battery and plug it into the pcb
        \item
    \item Send command to move into crawling configuration
    \item Place robot in the specified initial configuration
    \item Allow the system to stabilize for the specified duration
    \item Apply the command sequence
    \item Measure distance from start line to Node 1
    \item Stop the robot and save all recorded data
\end{enumerate}
